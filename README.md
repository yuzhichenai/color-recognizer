# 🎨 颜色识别器

上传图片，点击或框选区域，一键提取颜色 HEX 色号。

## ✨ 功能

- **图片上传** — 支持文件选择、拖拽上传、Ctrl+V 粘贴
- **点击取色** — 点击图片任意像素，立即获取颜色值
- **区域取色** — 拖拽框选区域，自动计算平均颜色
- **一键复制** — 点击复制 HEX 色号到剪贴板
- **取色记录** — 历史颜色列表，点击可重新选中

## 🔗 在线体验

[https://yuzhichenai.github.io/color-recognizer/](https://yuzhichenai.github.io/color-recognizer/)

## 🚀 使用方式

1. 上传图片（文件选择 / 拖拽 / 粘贴）
2. 选择取色模式：**点击取色** 或 **区域取色**
3. 在图片上点击或拖拽
4. 色号自动显示在右侧面板，点击「复制色号」即可
