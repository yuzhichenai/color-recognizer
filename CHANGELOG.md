# 更新日志

## v1.3.0 (2026-06-30)

### 🚀 新功能
- **多格式同时显示** — HEX / RGB / HSL 三行同时展示，每行右侧独立复制按钮
- **色彩搭配方案** — 取色后自动计算互补色、近似色、三角调和配色，点击色块即可复制
- **快捷键帮助面板** — 点击顶部 `?` 按钮或按 `?` 键弹出快捷键列表
- **复制格式选择** — 可直接复制 HEX / RGB / HSL 任意格式的色号
- **一键重置视图** — 底部新增重置按钮，缩放一键归零
- **区域取色精度可选** — 底部新增 3×3 / 5×5 / 7×7 采样网格，控制颜色敏感度
- **触摸设备支持** — Canvas 添加 touch 事件，平板和手机上可用手指取色

### 💄 体验优化
- 取色提示气泡修复：固定在画布中心偏下位置，不再随机飘动
- 右侧面板重排版，视觉更紧凑，信息密度更高
- 新增 `0` 快捷键：一键重置缩放

## v1.2.0 (2026-06-30)

### 🚀 新功能
- **localStorage 持久化** — 取色记录自动保存，刷新不丢失；恢复上次选中的颜色
- **拾色器放大镜** — 鼠标悬停时弹出 7×7 像素放大环 + 十字准星，精确到单像素
- **颜色名称识别** — 内建 160+ 常见颜色名称库，取色时显示最接近的中文/英文色名
- **双击重置缩放** — 双击画布区域将缩放恢复为 100%

### 💄 体验优化
- **鼠标追踪取色** — 状态栏实时显示鼠标指向的像素坐标 `(x, y)` 和颜色值
- **记录容量提示** — 徽标改为 `3/20` 格式，直观显示已用/上限

### 🐛 修复
- 区域取色拖拽时鼠标移出画布，选择框不再残留

## v1.1.0 (2026-06-30)

### 🚀 新功能
- **滚轮缩放** — 鼠标滚轮缩放图片，右下角显示缩放比例
- **颜色格式切换** — 点击 HEX/RGB 数值循环切换 `HEX ↔ RGB ↔ HSL`
- **键盘快捷键** — `Esc` 取消选择、`+/-` 缩放、`C` 复制色号
- **全屏拖拽** — 拖拽文件到窗口任意位置显示遮罩提示，松手即上传

### 💄 体验优化
- **Canvas 自适应** — 画布自动适配窗口大小，不再固定 800×600
- **取色记录管理** — 悬停显示 × 可单条删除，底部「清空全部」按钮
- **状态栏增强** — 显示图片尺寸、缩放比例、当前模式
- **空状态引导** — 显示 `Ctrl+V` / `Ctrl+C` 等快捷键提示
- **大图片性能** — 超过 4096px 的图片自动缩小离屏画布

### 🐛 修复
- Canvas CSS `width:100%` 与 JS 固定尺寸不匹配导致的拉伸变形
- `getCanvasCoords` 现在基于动态画布尺寸计算
