# 更新日志

## v1.6.3 (2026-07-01)

### 🐛 修复

- **取色音效双击触发且持续不停** — `playPickSound` 改为 async 并 `await audioCtx.resume()` 解决浏览器自动播放策略；添加 `osc.stop()` 终止持续发声
- **`isLocked` 未定义导致历史渲染崩溃** — 补充 `isLocked()` 函数定义，恢复 `renderHist()` 正常渲染
- **历史色块看不到色号** — 历史项添加 `data-hex` 属性及 `::after` 悬浮提示，悬停时显示 HEX 色号
- **有历史记录时导出失败** — `showExport()` 前置条件从 `!_curColor` 放宽为 `!_curColor && _history.length === 0`

## v1.6.2 (2026-06-30)

### 🐛 修复
- **取色音效双击才触发** — AudioContext 改为全局单例，首次用户点击时自动 resume，首次取色即发声
- **导出格式切换时退出模态框** — `showExport()` 末尾删除了 `toggleModal()` 调用，切换格式不再关闭弹窗
- **右侧面板取色记录被压缩** — `.history-list` 增加 `min-height: 36px`，历史区域不再缩至零

## v1.6.1 (2026-06-30)

### 🎨 亮色主题重新设计
- 配色从冷灰（`#f5f7fa`）改为暖象牙白（`#f8f6f0`），界面更柔和舒适
- 卡片边框从 `0.08` 提高至 `0.1`，面板层次更分明
- 主文字从纯黑改为深炭灰 `#2d2d2d`，降低刺眼感
- 二级文字改为 `#666`，提升可读性
- 色行、搜索框、配色卡背景统一为暖灰 `#f3f0ea`
- 模式切换背景改为 `#ece8e0`，按钮更清晰可见
- 棋盘格颜色改为主题感知（`--chk-a` / `--chk-b` 变量），亮色模式使用 `#fcfbf7` / `#ece8e0`
- 暗色模式所有界面不变

## v1.6.0 (2026-06-30)

### 🚀 新功能
- **亮色/暗色主题切换** — 顶部 🌓 按钮一键切换，偏好自动持久化
- **色盲模拟器** — 取色后自动显示红/绿/蓝色盲三种模拟预览，点击色块复制 HEX
- **全屏画布模式** — 按 `F` 键隐藏所有面板，画布最大化，再按 `F` 恢复
- **图像滤镜** — 亮度/对比度/饱和度三滑块，调整后实时影响取色和主色提取
- **拖拽色环复制** — 色环可拖拽到其他应用（Figma/PS 等），释放自动复制 HEX
- **取色音效** — 取色时播放短促提示音，底部 🔊/🔇 可开关

### 💄 体验优化
- **取色记录搜索** — 历史区域顶部搜索框，按色号实时过滤
- **多格式导出扩展** — 导出弹窗新增 JSON / CSV 格式
- **图片信息提示** — Canvas 底部显示图片文件名与尺寸
- **AGENTS.md** — 新增项目规则文件，规范发布流程

## v1.5.0 (2026-06-30)

### 🚀 新功能
- **多图片管理** — 上传多张图片后通过标签栏切换，每张图片独立维护缩放、取色记录；支持上传多张或批量拖拽
- **WCAG 对比度检查** — 取色后输入另一颜色 HEX，自动计算 WCAG AA/AAA 对比度评级（正文/大文本），实时显示通过/失败
- **调色板导出为图片** — 将取色记录导出为 PNG 色卡图，色块 + HEX 标注
- **URL 分享调色板** — 将当前取色记录编码为 URL 查询参数，一键复制分享链接
- **智能色彩建议** — 主色变化时实时计算互补 / 近似 / 三角配色方案
- **可拖拽平移** — 放大后直接拖拽画布平移查看细节，底部显示平移坐标

### 💄 体验优化
- **首次引导页** — 首次打开时显示功能导览浮层，可选择"不再显示"
- **性能优化** — 主色提取改用 requestIdleCallback 调度，不阻塞 UI 渲染
- **对比度检查自动联动** — 取色或微调后对比度面板自动更新
- **空状态智能提示** — 根据当前是否为收藏过滤模式显示不同文案

## v1.4.0 (2026-06-30)

### 🚀 新功能
- **吸管取色模式** — 新增吸管按钮，鼠标在图片上悬停即自动捕捉颜色、实时更新面板；按住 `Alt` 键临时切换吸管模式，松开恢复
- **图片主色提取** — 图片加载后自动提取 Top 8 主色调（颜色量化算法），展示色条并支持一键复制；可按 `S` 键或右键菜单手动提取
- **CSS/Tailwind/SCSS 导出** — 顶部 📦 按钮，取色记录一键导出为 CSS 变量、Tailwind v4 配置或 SCSS 变量，弹出代码框直接复制
- **取色收藏** — 每条取色记录支持星标收藏（★/☆），「清空全部」仅清除未收藏项，顶部过滤按钮可切换仅显示收藏
- **HSL 滑块微调** — 取色后通过 H / S / L 三个滑块手动微调颜色，色环和色号实时联动更新

### 💄 体验优化
- **右键菜单** — Canvas 上右键弹出迷你菜单：复制颜色 / 重置缩放 / 提取主色
- **主色提取进度** — 图片加载后自动提取并 Toast 提示完成
- **底部栏响应式折叠** — 小屏设备采样精度按钮自动隐藏，模式按钮文字缩短
